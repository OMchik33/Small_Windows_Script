# Small_Windows_Script

В репозитории небольшие скрипты для Windows, которые могут пригодиться в жизни.

1. **merge_video_audio.bat** - Скрипт с меню, смотрит в папку вокруг себя и предлагает выбрать видео, аудио для склейки в единый файл с конвертацией. Использует ffmpeg, если его нет - устанавливает.
2. **ffmpeg_video+img.bat** - Скрипт с меню, смотрит в папку вокруг себя и предлагает выбрать видео, картинку, и вклеить картинку перед видео длительностью в 1 кадр. Полезно для выкладывания видео в соцсети чтобы появилась "превьюшка" видео. Использует ffmpeg, если его нет - устанавливает.
